exago

Welcome!

Exago is a code quality tool that inspects your repository and reports on what could be improved. It lets you see what's potentially wrong directly in the code.

This tool is entirely free for use and open source, contributions are welcome!
Read more about how it works in this blog article.

A few example projects:

  • go-kit/kit
  • inconshreveable/log15
  • gin-gonic/gin
